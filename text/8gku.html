8時限目：マルチメディア

先生、おはようございます。きょうはマルチメディア？

みなさん、おはようございます。マルチメディアとして、今日は音声と映像をJavaScriptでコントロールしましょう。例えば、数当てゲームで正解したら結果により違う音が流れたり映像が流れたら楽しいですよね。

本当ですね！

音声をコントロール

では、次のコードを用意しましょう。

<audio id="audio1" src="audio.mp3"></audio>

これは1学期のときにやりました。contorolsつければコントローラが表示されたんですよね。

そうですね。では要素の取得を行いましょう。

const audio1 = document.getElementById("audio1");

OKです。このaudio1を操作するんですね。

そうです。早速操作してみましょう。

再生する

audio1.play();

一旦停止する

audio1.pause();

カウンタの値

巻き戻しをします。

audio1.currentTime = 0;

音量の値

音量を半分にします。

audio1.volume = 0.5;

速度の値

速度を1.5倍速にします。

audio1.playbackRate = 1.5;

以上をまとめて、操作用のボタンを用いればこのようになります。

<h1>オーディオ</h1>
<p><audio id="audio1" src="audio.mp3"></audio></p>
<p>
	<button onclick="audio1.play();">再生</button>
	<button onclick="audio1.pause();">停止</button>
	<button onclick="audio1.currentTime=0;">巻き戻し</button>
	<button onclick="audio1.volume -= 0.2;">音を小さく</button>
	<button onclick="audio1.volume += 0.2;">音を大きく</button>
	<button onclick="audio1.playbackRate += 0.5;">早く</button>	
	<button onclick="audio1.playbackRate -= 0.5;">ゆっくり</button>
</p>

<script>
	const audio1 = document.getElementById("audio1");
</script>

思ったよりすごく簡単だ…。

映像をコントロール

映像は、音声と同じ方法でコントロールできますので、このようになります。

<h1>ビデオ</h1>
<p><video id="video1" src="video.mp4"></video></p>

<p>
	<button onclick="video1.play();">再生</button>
	<button onclick="video1.pause();">一旦停止 </button>
	<button onclick="video1.currentTime=0;">巻き戻し</button>
	<button onclick="video1.volume -= 0.2;">音を小さく</button>
	<button onclick="video1.volume += 0.2;">音を大きく</button>
	<button onclick="video1.playbackRate += 0.5;">早く</button>	
	<button onclick="video1.playbackRate -= 0.5;">ゆっくり</button>
</p>

<script>
	const video1 = document.getElementById("video1");
</script>

先生、面白いです。
